# Helloworld_of_DeepLearning
Simple Neural network generalize the MNIST dataset with NumPy and Pandas only

# Structure
- 784->128->64->10 neuron structure
- Relu Activations of hidden layers
- SoftMax for final activation
- CrossEntropy as a loss function
- Mini batch SGD

# HyperParameters
- batch size 32
- lr = 0.1


# Results
- 99.9% Training accuracy
- 95.5% validation accuracy
